## 概要
このプロジェクトは、未経験からWeb開発を学ぶために作成した学習用プロジェクトです。

プロジェクトを通して、開発の流れや基本的な機能実装の経験を積むことを目的としています。

## 機能概要
- 一般家庭のキッチンにある食材の在庫管理（賞味期限管理）をするアプリ
- トップページ
- ![在庫一覧](screenshots/whole_page.png)
- 
- 食材名 + 賞味期限の登録 ⇒ 一覧表示
- 一覧に登録した食材項目の編集と削除
- 賞味期限までの残り日数に応じた色分け表示＆アラート表示機能
- レシピ検索機能
